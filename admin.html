☁️ 云端同步配置

配置 Cloudflare Worker 代理后，导出操作将直接推送到 GitHub 仓库，无需手动下载和 git push。

工作原理：
  1. Cloudflare Worker 持有 GitHub PAT，浏览器不接触 token
  2. 管理员通信密钥用于验证请求身份，防止他人调用 Worker
  3. 导出时数据流：浏览器 → Worker → GitHub API → 仓库 → Pages 自动部署
